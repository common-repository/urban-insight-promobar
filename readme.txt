Urban Insight Promobar is a simple promobar plugin which allows you to customize a dismissable promobar at the top of your WordPress website.

== Description ==

Urban Insight Promobar makes it easy to add an attractive and functional promobar to the top of your website. This dismissable announcement promobar displays HTML content and Dashicons on a customizable background color. Administrative users can schedule, enable, and reset the promobar display with intuitive plugin settings.

= Features =

The Urban Insight Promobar plugin comes with everything you need to create and manage your sitewide announcement: 

* Custom HTML content display
* WordPress Dashicons font icon selector
* Background color picker option
* Options to schedule promobar display with start/end date and time settings.
* User dismissal tracking with js-cookie
* Browser cookie expiration (set in days)
* Dismissal reset setting using roll browser cookie value
* Localhost development option

== Installation ==

Installing the plugin is easy—just follow one of the following methods:

= Install Urban Insight Promobar plugin from within Wordpress =

1. Visit the plugins page from your WordPress dashboard and select ‘Add New.’
2. Search for "Urban Insight Promobar."
3. Activate Urban Insight Promobar from your plugins page.

= Install Urban Insight Promobar Manually =

1. Visit the plugins page from your WordPress dashboard and select ‘Add New.’
2. Select the 'Upload' option and select 'Choose File.'
3. When the popup appears, select the urban-insight-promobar-x.x.zip file from your desktop. (The 'x.x' will change depending on the current version number.)
4. Follow the on-screen instructions to complete the upload.
5. Once uploaded, activate the plugin via the prompt. A message will confirm activation. 
6. Access a link to access promobar frontend settings on the confirmation page.

== Frequently Asked Questions ==

= Does your plugin account for the WP Admin Bar height? =

Yes, our plugin adds top padding to the body tag to ensure that the promobar does not overlap with the header content.
